1. Use your arrow keys to control the snake as it moves through the canvas.
  2. Create a pattern using the movements of the snake, and once you are content with the pattern, press the spacebar to save your design!
  3. Enjoy!
This project was inspired by the game snake. I used the prelimanary concept and design of the game to create a method of drawing generative art that was indivdual to each user that they could then save to their device
